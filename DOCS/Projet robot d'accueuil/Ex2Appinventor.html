Exemple 2 APP Inventor: CONNECTER UN SMATPHONE A LA CARTE ARDUINO


Ce document est un petit exemple de prise en main  destiné à vous aider dans votre apprentissage de App Inventor pour connecter le téléphone à la carte Arduino, ce qui vous permettra par sa suite de se servir du téléphone comme d'une IHM (Interface Homme Machine)

Avant de commencer l'exercice, lire les chapitres "ressources app inventor" et regarder les tutoriels donnés ainsi que le document PDF suivant:  document pdf d'aide plus complet

CONNECTER LE TELEPHONE A LA CARTE ARDUINO

Configuration

Programme de l'émetteur (carte Arduino)

Programme : 1_arduino_bluetooth

Commentaires

//Initialisation des E/S et communication

void setup()

{

Serial.begin(9600);

}

//Programme principal

void loop()

{

 Serial.println("BONJOUR de");

delay(1500);        

Serial.println("prenom"); // Ecrire votre prénom dans la parenthèse

delay(1500);

   }

//Fin


Configuration débit  de transmission Bluetooth


Envoi cadencé à 1,5s  des messages

  • BONJOUR de
  • prénom


Raccordement du module bluetooth sur la carde arduino  

             


Identification bluetooth : itead ; code appariement : 1234
